🔎 PROJECT TITLE: AlgoLens
📺 TAGLINE: The Netflix for Code. Stop guessing—rewind, visualize, and simulate.

🚀 [SHORT DESCRIPTION - 150-200 characters]
AlgoLens is a time-travel debugger and algorithm visualizer that automates manual dry runs. Scrub through execution history and simulate 'What-If' logic changes in real-time.

😫 [PROBLEM STATEMENT]
Developers and students spend 70% of their time on manual "dry runs" using pen and paper. Standard debuggers are like Live TV—if you miss a moment, you have to restart the entire program. They show you "WHAT" broke (the crash), but never "WHY" it happened (the journey), leading to massive cognitive load and wasted hours of reproduction time.

💡 [THE SOLUTION]
AlgoLens provides the "God View" of code execution. By recording every state change and parsing the Abstract Syntax Tree (AST), it transforms static code into an interactive, 4K execution timeline. It allows users to:

⏪ Rewind and replay execution via a bidirectional seek bar.

🌳 Visualize complex data structures (Trees/Graphs) automatically.

🌌 Test fixes in "alternate realities" using Mid-Execution Injection.

🛠️ [CORE FEATURES]

⏪ Temporal Scrubbing: Bidirectional seek bar to navigate execution history.

🌳 AST-Native Visualization: Automatic rendering of logic-first data structures.

🌌 The "What-If" Multiverse: Change variable values mid-execution to simulate new outcomes instantly.

🤖 AI "WHY" Panel: Gemini-powered natural language explanations of logic failures.

💻 [TECH STACK]

⚛️ Frontend: React.js, Tailwind CSS, Framer Motion

⚙️ Engine: WebAssembly (WASM) for high-speed state diffing

🌳 Parsing: Acorn / Escodegen (AST Manipulation)

🧠 Intelligence: Google Gemini API (Natural Language Reasoning)

💎 [UNIQUE SELLING POINTS (USP)]

🔄 Bidirectional Navigation: Unlike VS Code, we go backward as easily as forward.

✨ Zero-Replay Reproduction: Test logic fixes without restarting the process.

🎨 Intent-Aware UI: We visualize the "Algorithm," not just the "Memory Addresses."
